tHqAnalysis
=======

Framework of the tHq group of Karlsruhe, much credit to the ttH group

## Installation
Follow These Steps:

    cmsrel CMSSW_7_2_3
    cd CMSSW_7_2_3/src
    cmsenv

    git clone https://github.com/cms-ttH/MiniAOD.git
    git clone https://github.com/sfink/tHqAnalysis.git
    
    scram b -j 12
