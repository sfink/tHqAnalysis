tHqAnalysis
=======

Framework of the tHq group of Karlsruhe, much credit from the ttH group

## Installation
Follow These Steps:

    cmsrel CMSSW_7_2_3
    cd CMSSW_7_2_3/src
    cmsenv

    git clone https://github.com/sfink/MiniAOD.git
    git clone https://github.com/sfink/tHqAnalysis.git
    
    scram b -j 12
